# Numbers

- int
- float
- complex

## Type

[numbers.py](numbers.py)

## Conversion

```py
int_to_float = float(35)
float_to_int = int(44.2)
int_to_complex = complex(35)
print(int_to_float)
print(float_to_int)
print(int_to_complex)
print(type(int_to_float))
print(type(float_to_int))
print(type(int_to_complex))
```
